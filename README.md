# kodumisto
A code editing bot

April 2, 2023 Explainer Video: https://www.youtube.com/watch?v=DpFIE-7skyA

Test Issues are in `kodumisto-playground` repo: https://github.com/busse/kodumisto-playground/issues

> Kodumisto (Esperanto): Derived from the Esperanto words "kodo" meaning "code" and "umisto" meaning "skilled person," this name suggests the bot's expertise and competence in editing software code.
> - ChatGPT-4


## Setup

1. Install `requests`

> `pip install requests`

2. Install `langchain`

> `pip install langchain`


3. Install `PyGithub`

> `pip install PyGithub`

4. Install `python-dotenv`

> `pip install python-dotenv`
